# edih-analitika
**EDIH-Analitika** je Streamlit aplikacija razvijena za analizu i izvještavanje u okviru projekata Europskih Digitalnih Inovacijskih Hubova (EDIH). Aplikacija omogućuje: - automatsku obradu DMA PDF izvještaja, - AI OCR ekstrakciju i sažimanje (GPT-4o-mini), - interaktivne grafove i tablične preglede, - keširanje i generiranje agregiranih analiza.
